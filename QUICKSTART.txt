STEP 1: Install Docker (if not already installed)
  $ curl -fsSL https://get.docker.com -o get-docker.sh
  $ sh get-docker.sh

STEP 2: Login to Docker Hub
  $ docker login
  # Enter your Docker Hub credentials

STEP 3: Build the Docker image
  $ cd /opt/echomimic_v2_deploy
  $ docker build -t YOUR_DOCKERHUB_USERNAME/echomimic-v2:latest .
  # This will take 10-15 minutes

STEP 4: Push to Docker Hub
  $ docker push YOUR_DOCKERHUB_USERNAME/echomimic-v2:latest

STEP 5: Deploy to RunPod
  A. Go to: https://www.runpod.io/console/serverless
  B. Click New Endpoint
  C. Fill in:
     - Name: echomimic-v2
     - Container Image: YOUR_DOCKERHUB_USERNAME/echomimic-v2:latest
     - Container Disk: 40 GB
     - GPU: NVIDIA A40
     - Min Workers: 0
     - Max Workers: 3

STEP 6: Copy your endpoint credentials
  After deployment, RunPod will give you:
  - Endpoint ID (e.g., abc123def)
  - API Key (e.g., rpa_XXX...)

STEP 7: Update VIDS Pro configuration
  $ ssh root@135.181.99.172
  $ nano /opt/vids_pro/.env
  
  Uncomment and update these lines:
  RUNPOD_ECHOMIMIC_KEY=rpa_YOUR_KEY_HERE
  RUNPOD_ECHOMIMIC_RUN=https://api.runpod.ai/v2/YOUR_ENDPOINT_ID/run
  RUNPOD_ECHOMIMIC_STATUS=https://api.runpod.ai/v2/YOUR_ENDPOINT_ID/status

STEP 8: Restart VIDS Pro backend
  $ systemctl restart vids-pro-backend
